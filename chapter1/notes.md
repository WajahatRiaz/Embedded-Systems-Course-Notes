## Introduction to Embedded Systems
What is an **Embedded Systems**? <br>
A computer system embedded within a real world application. It is usually real-time in nature. Modern ES have 
- Application specific hardware ASICs, FPGAs, etc;
- Programmable processor;
- Transducers and actuators.


### Characteristics of a real time system
- Embedded systems respond to events<br>
- An embedded system has tasks which are real-time
- Results must be correct within the stipulated time
- Safety and criticality
- Concurrency
- Distributed and feedback
- Custom Hardware
- Reactive
- Stability
- Exception handling
- Saftey and Reliability
